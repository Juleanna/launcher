🚀Быстрый старт

Минимальные требования системы:

ОС: Windows 7/10/11, Linux, macOS
Python: 3.7 или новее
ОЗУ: 512 МБ
Диск: 150 МБ + место для игр

Первый запуск

  1. Скачайте лаунчер в папку по вашему выбору
  2. Откройте командную строку в папке лаунчера
  3. Установите зависимости: pip install -r requirements.txt
  4. Запустите лаунчер: python Launcher.py
При первом запуске лаунчер автоматически создаст необходимые файлы конфигурации и настроит систему безопасности.

🎮Основные функции

⚡Быстрые загрузки

Delta-обновления экономят до 70% трафика. Возобновляемые загрузки с паузой и автоматическим выбором зеркал.

🔒Безопасность

RSA цифровые подписи, SHA-256 проверка целостности, принудительное HTTPS шифрование.

🌐P2P сеть

Peer-to-peer распределение нагрузки между пользователями для ускорения загрузок.

📊Статистика

Детальная аналитика скорости, использования сети, P2P метрик и истории операций.

💾Резервные копии

Автоматические бэкапы с возможностью отката к предыдущей версии при ошибках.

🔄CDN поддержка

Автоматический выбор ближайшего зеркала и переключение между CDN серверами.

🌐Веб-контент и новости

Лаунчер поддерживает отображение веб-контента прямо в интерфейсе через систему табов.

Настройка источников новостей

[WebContent]
auto_refresh = 1          # Автоматическое обновление
refresh_interval = 300    # Интервал в секундах (5 минут)

[NewsSource1]
name = Новости сервера    # Название таба
url = https://yourserver.com/api/news.json
type = json
enabled = 1

[NewsSource2]
name = Обновления
url = https://yourserver.com/rss/updates.xml
type = rss
enabled = 1

📦Система обновлений

Генератор обновлений (Update.py)

  1. Запустите python Update.py
  2. Введите номер версии (например: 1.2.3)
  3. Выберите директорию с новыми файлами
  4. Выберите выходной файл
  5. Настройте безопасность (цифровые подписи)
  6. Настройте Delta-обновления (опционально)
  7. Нажмите "Сгенерировать файл обновлений"
Delta-обновления экономят 50-80% трафика! Они содержат только изменения между версиями.

🔧Устранение неполадок

❌Лаунчер не запускается

# Проверьте Python
python --version

# Переустановите зависимости
pip install --force-reinstall -r requirements.txt

# Проверьте права доступа
# Windows: запуск от имени администратора

🐌Медленная загрузка

  • Включите P2P: enable_p2p = true
  • Увеличьте соединения: max_connections = 16
  • Проверьте CDN зеркала
  • Отключите VPN/антивирус

🔐Ошибки подписей

# Временное отключение (не рекомендуется)
[Security]
verify_signatures = false

# Или обновите ключи с сервера

🌐P2P не работает

  • Откройте порт в роутере/фаерволле
  • Проверьте трекер-сервер
  • Убедитесь что есть другие пиры
  • Проверьте сетевые настройки

Полезные команды диагностики:

# Проверка зеркал CDN
python -c "from cdn_manager import get_cdn_manager; asyncio.run(get_cdn_manager().check_all_mirrors())"

# Тест P2P
python -c "from p2p_distribution import P2PDistributor; print('Peer ID:', P2PDistributor().get_peer_id())"

# Статистика сети
python -c "from bandwidth_optimizer import get_network_optimizer; print(get_network_optimizer().get_statistics())"
